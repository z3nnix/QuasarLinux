## Как это работает

### 1. Определение страты

Страта — изолированный слой с собственным пакетным менеджером и пакетами. Пример: одна страта это Arch с pacman, другая — Debian с apt.

При запуске Quasar поднимает все настроенные страты (Arch и Debian). Они видят общую файловую систему, но не влияют друг на друга без явной команды пользователя.

### 2. Выбор источника пакета

- `qsr install firefox`     -> установка из Arch (основная страта по умолчанию).
- `qsr install deb:firefox` -> установка из Debian через apt.

Автоматический выбор пакета по наличию или версии в текущей версии QSR не реализован, но планируется. Ручное указание `deb:` гарантирует предсказуемость.
