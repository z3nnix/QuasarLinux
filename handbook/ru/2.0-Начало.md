# Системные требования:

| Компонент | Минимум | Комфорт |
|-----------|---------|---------|
| ОЗУ | 1 ГБ | **4+ ГБ** |
| Диск | 8 ГБ | **32+ ГБ** |
| CPU | 1 ядро, 1 ГГц | 2+ ядра, 2+ ГГц |

_но вообще — Quasar далеко не самый подходящий дистрибутив для слабых ПК, увы_

# Установка.
> Предисловие: В Quasar установка двух-этапная: до перезагрузки(из под quasariso) и после(пост-инсталяция). Сумарно будет два ребута, это нормально.
## Этап 1.
Для начала, [установите iso-образ](https://github.com/z3nnix/QuasarLinux/releases/tag/release). После, создайте загрузочную флешку с этим iso и перезагрузитесь в флешку.

### Когда вы уже в quasariso:
Первым делом, проверьте подключение к интернету:
```bash
ping google.com
```
Если все хорошо, то спускайтесь ниже, если нет -> проблема скорее всего в том, что вы используете именно WiFi. Легко гуглиться по запросу в духе _Не работает WiFi в archlinux_

Когда с доступом в интернет все будет в порядке, начните установку:
```bash
# ВАЖНО! на последнем экране quasarinstall ОБЯЗАТЕЛЬНО нужно будет выбрать
# простой выход из инсталлера. Это нужно для корректного исполнения скрипта,
# прокидывающего post-install скрипт в ново-установленную систему

quasarinstall
```
_На самом деле, это просто archinstall с другим брендингом. Он простой, вы обязательно разберетесь :)_

После, выполняйте перезагрузку.

## Этап 2.
Вас встретит брендинг.. как ни странно archlinux. Дело в том, что установка не завершена.

Войдите как superuser(root) и введите одну команду:
```bash
quasarify # и следуйте дальнейшим инструкциям
```

После -> перезагрузка и вас встретит QuasarLinux!!!
