# Первые шаги после установки
Пожалуй, лучшее, что можно сделать сразу после установки Quasar это на всякий случай обновить его(это не займет много времени):
```bash
sudo qsr update # заодно проверите, установился ли QSR вообще (если нет — см. страницу 6.0)
```

_Вообщем.. дальше вы вправе делать со своей системой что угодно ибо установка завершена_

# Что такое QSR?
QSR (Quasar System Resource) — это мета-пакетный менеджер. Он не заменяет pacman или apt, а управляет ими. Когда вы вводите `qsr install cowsay`, QSR сам решает, из какой страты (Arch/Debian/другой) взять пакет. Подробнее см. на странице 4.0.

## Примеры команд:
Вы можете посмотреть, какие команды существуют в qsr:
```bash
qsr
```

Можете установить какой-нибудь пакет:
```bash
sudo qsr install cowsay
```

И посмотреть состояние системы:
```bash
sudo qsr doctor
```
