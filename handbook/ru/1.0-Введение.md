# Обзор
Quasar — это мета-дистрибутив, сочетающий в себе несколько других дистрибутивов (элегантно).

## Проблема
в Arch нет какого-то пакета? В AUR он есть, но собирать из исходников — лень и риск. В Debian пакет есть, но чтобы его поставить, нужно или переходить на Debian целиком, или мучиться с контейнерами.

**Решение**: основная система Arch (свежий, привычный), но если пакета нет в Arch — QSR автоматически возьмёт его из Debian. Никакой возни с AUR, никаких контейнеров. Просто `qsr install <пакет>`.

## Для кого этот дистриубтив?
Quasar будет идеальным дистрибутивом для тех, кто:
- Желает абстрагироваться от излишней фрагментации Linux
- Желает иметь доступ к бóльшей части экосистемы Linux
- Не страшиться преград

## А если я хочу Debian как основу?
Из коробки Quasar использует Arch, потому что большинству нужны свежие пакеты. Но вы можете изменить приоритет страт. (пример **будет** в документации.)

Сделайте Debian основным и получите стабильную базу с возможностью ставить свежие пакеты из Arch. Quasar не запрещает, просто не делает этого по умолчанию.
