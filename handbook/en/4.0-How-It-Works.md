## How It Works

### 1. Stratum definition

A stratum is an isolated layer with its own package manager and packages. Example: one stratum is Arch with pacman, another is Debian with apt.

When Quasar boots, it brings up all configured strata (Arch and Debian). They share the same filesystem but don't interfere with each other without an explicit user command.

### 2. Package source selection

- `qsr install firefox`     -> install from Arch (default primary stratum).
- `qsr install deb:firefox` -> install from Debian via apt.

Automatic package selection based on availability or version isn't implemented in current QSR, but it's planned. Manual `deb:` prefix guarantees predictability.
