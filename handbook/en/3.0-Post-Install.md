# First steps after installation
Probably the best thing to do right after installing Quasar is to update it, just in case (it won't take long):
```bash
sudo qsr update # also checks if QSR is installed at all (if not — see page 6.0)
```

*Anyway... from here you're free to do whatever you want with your system, because the installation is complete*

# What is QSR?
QSR (Quasar System Resource) is a meta-package manager. It doesn't replace pacman or apt — it manages them. When you type `qsr install cowsay`, QSR decides which stratum (Arch/Debian/other) to take the package from. See page 4.0 for details.

## Command examples:
You can see what commands exist in qsr:
```bash
qsr
```

You can install some package:
```bash
sudo qsr install cowsay
```

And check system health:
```bash
sudo qsr doctor
```
