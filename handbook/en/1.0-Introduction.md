# Overview
Quasar is a meta-distribution that elegantly combines several other distributions.

## The Problem
Need a package that's not in Arch? It's probably in AUR, but building from source is a chore and a risk. The package exists in Debian, but to get it you'd either switch entirely to Debian or wrestle with containers.

**The Solution**: Your base system stays Arch (fresh, familiar). But if a package isn't in Arch — QSR automatically pulls it from Debian. No AUR hassle, no containers. Just `qsr install <package>`.

## Who is this distro for?
Quasar is ideal for those who:
- Want to abstract away from Linux's excessive fragmentation
- Want access to a larger part of the Linux ecosystem
- Aren't afraid of a few bumps in the road

## What if I want Debian as my base?
Out of the box, Quasar uses Arch, because most people want fresh packages. But you can change the stratum priority (example **will be** in the docs).

Make Debian your primary and get a stable base with the ability to install fresh packages from Arch. Quasar doesn't forbid it, it just doesn't do it by default.
