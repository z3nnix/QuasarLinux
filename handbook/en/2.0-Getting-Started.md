# Installation
> Foreword: Quasar installation is two-stage: before reboot (from quasariso) and after (post-install). Two reboots total, that's normal.

## Stage 1
First, [download the ISO image](https://github.com/z3nnix/QuasarLinux/releases/tag/release). Create a bootable USB with this ISO and reboot into the USB.

### Once you're in quasariso:
First, check your internet connection:
```bash
ping google.com
```
If all good, proceed below. If not — the issue is most likely WiFi. Easily googlable with something like "WiFi not working in archlinux"

Once you have internet access, start the installation:
```bash
# IMPORTANT! On the final screen of quasarinstall you MUST select
# "exit installer" (simple exit). This is needed for the post-install script
# to be properly passed into the freshly installed system

quasarinstall
```
*It's actually just archinstall with different branding. It's simple, you'll figure it out :)*

After that, reboot.

## Stage 2
You'll be greeted by... archlinux branding, surprisingly. That's because the installation isn't finished yet.

Log in as superuser (root) and run one command:
```bash
quasarify # and follow further instructions
```

After that -> reboot and QuasarLinux will greet you!!!
